KITTY RUN - A dirty frogger rip-off

This game is inspired by Frogger, if someone is old enough to remember. The main idea has been borrowed from the original: an animal crossing a street and river, trying not to get hit by vehicles or drown, and reach a home. My take on the original game features a cat instead of a frog, different obstacles as well as floating items on the river, and some other simplifications adaptions.

I tried to keep the game "retro" in some way, fancy graphics and stuff wouldn't have done the original game any justice (and would've been beyond my programming skills anyways...).

This is the first project of a Web-Development Bootcamp at Ironhack, Berlin, March 2020.
